# 漢字学習学年判定ツール

入力された漢字が、小学校の何年生で習うもの（学年別漢字配当表）かを判定して出力するツールです。

## 概要
文部科学省の学習指導要領に基づき、漢字の習得学年を特定します。
教育現場での資料作成や、お子様の学習状況に合わせたテキストの選定などに活用いただけます。
